检测鼠标输入

以下内容在Update中使用

0 左键; 1 右键; 2 中键

检测按键输入

  • 检测按键还有个传入字符串的重载, 但是只能传入小写字母的字符串, 不建议使用
  • 可以进入KeyCode中查看可以检测的按键

检测默认轴输入

在ProjectSettings中的InputManager可以设置轴输入, 也可以添加自定义的轴

  • GetAxis和GetAxisRaw的使用方法相同, 只不过GetAxisRaw只会返回-1,0或者1, 不会有值的渐变; 而GetAxis在按键变化时值会有渐变
