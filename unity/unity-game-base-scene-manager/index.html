Unity游戏基础框架6 -- 场景管理模块

发表于 2021-03-27 分类于 Unity 阅读次数： Valine：

场景管理模块用于统一管理场景的切换

可以在场景切换前后利用事件管理模块发送 场景退出 和 场景进入 的事件, 也可以在异步加载时发出 加载进度 的事件

需要切换的场景Scene要提前添加到BuildSettings中的场景列表中, 否则不能切换
