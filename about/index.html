Update 2020-01-07

我的理想是, 成为一个精英开发者, 开发出一款成功的游戏, 挣很多money, 成立自己的小团队

  • 打牢基础, 一个优秀的开发者, 一定基础扎实
    • 算法 熟练掌握, 应用
    • C++ 面向对象, 泛型, 现代C++(了解新特性), 高效的做法(effective), 标准库
    • 计算机网络
    • 现代操作系统
    • 设计模式
  • UE4蓝图 & C++开发, 感兴趣, 就放开了干, 拼命的学
    • 基础操作 了解基本概念(编辑器, 灯光, 材质, 地形, 粒子)
    • 蓝图学习 常用接口, 事件/函数/宏
    • 功能性学习 人物控制/技能释放/伤害判定/边缘高亮/AI(行为树等)…
    • 实例学习 跟着教程做一些小的案例, 实战中理解, 多做笔记
    • UE4 C++ 体系学习 继承结构, 常用类的作用(GamePlay框架, GameMode, Actor等)
    • 制作一个像样的游戏
  • 支撑技能
    • Git
    • Go & Python
    • 数据 (Redis, SQLite, MySQL)
    • Jenkins
    • 正则表达式
    • Linux
  • 机器学习(这个和游戏开发暂时不怎么相关), 不管泡沫多大, 它依然是现在的潮流, 学会了说不定能制作一款能自主学习的游戏, 多多学习, 有益无害
    • 引入机器学习的游戏制作
  • 坚持写blog, 记录成长的点滴

写个鬼的About页, 根本就不想这样思考, 也不敢承认自己.

不知道怎样描述自己? 并不是的, 只是不想去面对和承认现在的自己, 心中的执剑少年屠龙勇士与现在碌碌无为疲于奔命的打工仔差的太远太远, 平庸的我和父母妻子期待中的人中龙凤根本不能同日而语.

山村出生的我最能体会到爸妈拉扯一个大学毕业生是多么不易, 大半辈子的打拼, 我不知道该怎样描述, 但毕业后他们那明显霜染的鬓发说明了一切. 之前厌学不想上学的时候他们说”等你大学毕业了就不管你了”, 我还天真的觉得自己终于要自由了, 五年的大学时间, 半玩半学半混, 潇潇洒洒, 答辩后拿到厚厚的毕业证书在心中暗自高兴, 原来大学毕业证书这么好搞. 没有想过这么久的懒散让我失去了什么, 没有想过他们拿半辈子期待的”高材生”和拿着毕业证书的废柴的区别, 看到别人在算法竞赛中斩获一等奖我觉得哇真是天赋异禀.

看到有人去了理想的公司, 有人在喜欢的方向小有所成, 开始后悔, 要是大学好好学习就好了, 要是那时候我也挑一个方向认真钻研, 现在说不定也能找到好工作; 早知道实习这么重要, 当初暑假的时候就不去学校安排的实习了, 自己找个好点的就好了; 工作了, 这根本没时间学习呀, 哎玩的时间都没有哪有时间学.

然后想到未来时, 期待自己成为了精英开发者, 开发一款成功的游戏, 挣到很多money, 还有自己的小团队布拉布拉.

后来受到点打击, 又觉得自己啥也干不了, 改变不了, 觉得梦想遥不可及, 自己想太多了, 还是老老实实当个普通人.

我忘了吗? 当初最不想的就是成为一个啥都没做成的平庸普通人, 为什么会想成为自己最不想成为的人? 这个梦想有这么难, 难到10年20年一辈子也不能达到? 为什么会觉得自己想多了? 为什么想放弃的时候不想想爸妈的付出和期待, 为什么没想想婚礼上说的一定要给她幸福的生活?

其实心里清楚, 就自己上面说的那个梦想”成为一个精英开发者, 开发一款成功的游戏, 挣到很多money, 成立一个小团队”, 要是真的全力以赴, 怎么可能10年都达不到, 怎么可能作为一个一辈子的目标还遥不可及? 主要还是做的太少了, 实在太少了, 想这想那, 想一大堆, 什么都没做, 什么都没做还想要这想要那才显得距离那么远, 假设这个目标是一个距离我100km甚至1000km, 10000km的地点, 听起来很远, 但是每天跑步不都可以跑出4km吗? 如果我没有在路上想这么多, 只管朝着自己期望的方向不停的前进, 动起来, 结果会怎么样?

在此认清自己, 列出目标, 我希望自己不要想那么多, 不要犹豫不决, 出发都不敢, 多落到实处, 一直朝目标前行, 每次更新时都离梦想更近一点. 每个月更新一次都能看到变化.

每个人的成功肯定不是天赋异禀那么简单, 好搞的东西一般不会怎么有价值, 真正珍贵的事物一定来之不易.

希望自己能不忘初心, 方得始终. 希望自己能不负父母妻子厚望, 不让他们的付出和信任白费. 希望成为自己心中的英雄, 执剑少年.

Update 2019-11-25 想到的就写一下

我是一个怎样的人?

迟迟没有写这一篇, 想来想去竟然不知道怎样描述自己.
