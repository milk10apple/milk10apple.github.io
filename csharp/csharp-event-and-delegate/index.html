C#中的事件与委托实例的区别

发表于 2020-11-13 更新于 2021-01-23 分类于 C# 阅读次数： Valine：

区别概括为两点:

  1. 允许的操作: 事件只允许通过 += 和 -= 来订阅和取消订阅; 但是委托实例除了+=和-=, 还可以使用=直接赋值操作

  2. 调用的位置: 事件只能在所在类的内部调用(事件的调用也称做触发), 而委托实例在类外部也可以调用

为了便于区分, 本文中将委托和委托实例区分称呼

委托是一种自定义类型, 而委托类型的变量我们称为委托实例
